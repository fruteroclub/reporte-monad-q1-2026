# 🍌 Monad Blitz CDMX — Reporte Frutero Club

Reporte de actividades Frutero Club durante **Monad Blitz Ciudad de México**, realizado el 21 de febrero de 2026.

## 📊 Contenido del Reporte

- **Overview General** — Métricas clave del evento
- **🍌 Ana Banana** — Primera iniciativa de agente DevRel IA en hackathon presencial
- **Proyectos** — 18 submissions aprobados con detalles completos
- **Convocatoria** — Alcance en redes sociales (13,530 impresiones, 309 likes)
- **Mentoras** — Equipo de 6 mentoras especializadas

## 🚀 Stack Técnico

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Charts:** Recharts
- **Deployment:** Vercel

## 📦 Instalación

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 🍌 Ana Banana — Innovación Destacada

Por primera vez, Frutero Club desplegó un agente DevRel IA dedicado para soporte técnico en tiempo real durante un hackathon presencial. Ana Banana ofreció:

- Setup de entorno (Foundry, wallets, testnets)
- Arquitectura de smart contracts
- Integración con Monad testnet
- Resolución de pivotes técnicos bajo presión

Repositorios públicos:
- `fruteroclub/monad-blitz-agents` — Configuración reutilizable
- `fruteroclub/monad-blitz-starter` — Template para builders
